# Examples

These tests provide some examples of how to use the PQC-algorithms offered by the 
.NET platform and a test of the examples shown in the readme.

## KemBasedNet.Examles

See folder &lt;KemBasedNetTest&gt;/KemBasedNet/Examples

| Class | Purpose |
| --- | --- |
| ExamplesMLKem | Shows the usage of [ML-KEM](https://csrc.nist.gov/pubs/fips/203/final) (native PQC key exchange algorithm) using temporary or persistent private keys |
| ExamplesMLDsa | Shows the usage of [ML-DSA](https://csrc.nist.gov/pubs/fips/204/final) (native PQC signature algorihm) |
| ExamplesCompositeMLDsa | Shows the usage of [CompositeMLDsa](https://lamps-wg.github.io/draft-composite-sigs/draft-ietf-lamps-pq-composite-sigs.html) (combined PQC & traditional signature algorithm) |
| TestReadmeExamples | Tests for examples shown in the readme. |
